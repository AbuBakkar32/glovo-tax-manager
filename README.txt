Glovo Rider - Regime Forfettario Tax Manager
=============================================

Kibhabe install ebong run korben:

1. Prathame computer-e Node.js install thakte hobe.
2. Zip file ti unarchive/extract korun.
3. Terminal athoba Command Prompt open kore extracted folder-e jaan.
4. Nicher command ti run korun dependencies install korar jonno:
   npm install

5. Ebar local server ti start korun:
   npm start

6. Apnar jekono browser open kore jaan:
   http://localhost:3000

Dashboard e data add korle automatic "data.xlsx" file e save hobe.
